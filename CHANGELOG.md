# Change Log

All notable changes to the "Terminal System Notifier" extension will be documented in this file.

## [0.0.1] - 2026-01-29

### Added
- Initial release of Terminal System Notifier
- Terminal monitoring functionality with configurable check intervals
- AI-powered notification decision making
- Support for OpenAI, Claude, and custom API endpoints
- State machine to prevent duplicate notifications
- Configurable cooldown periods between notifications
- Commands for enabling/disabling monitoring
- Command to manually check all terminals
- Command to clear state machine
- Comprehensive configuration options
- Shell integration for terminal content capture

### Features
- Automatic terminal activity monitoring
- Smart notification system using AI analysis
- Multi-terminal support
- Notification cooldown to avoid spam
- Configurable minimum content length
- Support for multiple AI providers
